Estilos Locais

Método inline X método local

Diferente do método inline onde HTML e CSS se misturam, no método local, estilo e conteúdo ficam bem separados.
Para utilizar o método local, digite após a tag <title>, dentro de <head>, a referência da tag que você quer estilizar (ex: "body") sem colchetes angulares e digite uma chave simples. Em seguida é só inserir os parâmetros que deseja estilizar (ex: background-color, font-family, font-size, etc.).

Desvantagens do método

Essa técnica tem algumas desvantagens. Primeira: a parte de estilo ocupa uma grande parte do seu código. Segunda: caso seu cliente tem diversas páginas web, você deverá copiar e colar todas as linhas de estilo em todas as páginas.
Podemos concluir que o método local é vantajoso para quando nosso cliente solicita apenas uma única página web.
