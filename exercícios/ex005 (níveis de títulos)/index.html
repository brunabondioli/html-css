Os intrumentos de gestão

"h1" é a tag utilizada para títulos de primeiro nível. Há uma lenda que as páginas de internet não podem ter mais do que um título de nível 1, mas isso já foi desmentido (ver no material de apoio)

"h2" é a tag utilisada para títulos de segundo nível, como podemos ver!

"h3" é a tag utilizada para títulos de terceiro nível.

"h4" por fim é a tag utilizada para títulos de quarto nível.

O nível máximo de texto é "h6".
