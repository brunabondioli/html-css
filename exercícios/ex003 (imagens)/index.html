Testando carga de imagens

Abaixo você vai ver uma imagem que está na mesma pasta.

Para inserir uma imagem, basta começar a digitar "img" e dar enter. "Scr" significa source ou origem. Entre suas aspas, apertamos ctrl + espaço para abrirmos uma pequena lista pop-up das imagens que estão na pasta desse arquivo HTML. Aí, é só selecionar a imagem desejada.
Já, "alt" significa texto alternativo. Entre as aspas desse espaço, você deve escrever um texto para explicar a imagem (acessibilidade).

Lembre-se de deixar a imagem em um tamanho adequado para não sobrecarregar seu site.

Cuidado também ao mudar o nome dos arquivos de imagem na pasta do Explorer. Se você não mudar igualmente no código, a imagem não aparecerá (somente o texto alternativo).

Podemos também carregar imagens que estão em sub-pastas. Para isso, dentro das aspas de src, digite ctrl + espaço e busque na lista a sub-pasta onde a imagem desejada está. Então, clique duas vezes sobre essa pasta e selecione o arquivo de imagem desejado.

Também podemos carregar imagens externas. Procure a imagem na internet (Google Imagens), abra o site em que ela está inserida e clique com o botão direito do mouse. Depois clique em "copiar endereço da imagem" e colar (crtl + v) dentro das aspas de src. E voilá:

Entretanto, lembre-se que como esta imagem está em outro servidor, se ela cair nesse servidor, cairá no seu site também.
