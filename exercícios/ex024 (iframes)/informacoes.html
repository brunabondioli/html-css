Informações sobre iframes

É importante entender que 'iframes' são diferentes de 'frames'. O primeiro, apesar de apresentar problemas de segurança, ainda é bastante utilizado (YouTube?), já o segundo realmente foi desprezado pela comunidade front-end.

O Google utiliza um robô. O GoogleBot para verificar quais sites têm mais relevância.O GoogleBot apresenta problemas para indexar conteúdos de iframes. Não use iframes para conteúdos muito importantes.

Os iframes também não são recomendados para pessoas que necessitam de acessibilidade. Os softwares leitores de tela possuem problemas para acessar iframes.

Por fim, existem os problemas de usabilidade do navegador e do próprio usuário (por exemplo, voltar para a página anterior do site do iframe e sites não responsivos).

Os iframes abrem uma porta no seu site para outros sites. Isso pode ser pouco seguro (cuidado para quem você abre a porta). Podem ocorrer vazamentos de dados/ataques de xss e cross-site.
