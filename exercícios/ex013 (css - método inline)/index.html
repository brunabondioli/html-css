Estilos Inline

Basicamente dentro de cada tag podemos inserir estilos inline.
Para mudar a cor do plano de fundo da página, por exemplo, adicione dentro da tag <body> o parâmetro "style=background-color".
Já para mudar a fonte, adicione o parâmetro "font-family" em seguida, logo após o ponto-e-vírgula.
Para mudar a cor da fonte, digite dentro da tag <h1> o parâmetro "style=color"! Veja:

O método de estilo inline é útil, mas deve ser evitado no dia a dia. Para alterações pontuais, ele até serve, mas para dar estilo ao seu site, é inviável, já que ele é trabalhoso quanto à manutenção e polui seu código.
