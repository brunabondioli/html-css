Estilos Externos

Mais versátil

O método de estilos externos é o mais versátil e eficiente, especialmente quando temos muitas páginas em um site.
Para utilizá-lo, digite "link:css" logo após a tag <title> e dentro de "href" aceite a sugestão de abrir o arquivo "style.css". Como o arquivo ainda não existe, segure a tecla "Ctrl" e clique no link do aqruivo para criá-lo.
Dentro do arquivo "style.css", cole todas as linhas referentes ao estilo da sua página. Não se esqueça de:

  • digitar na primeira linha do arquivo css a regra "@charset "uft-8""
  • colocar o link para o arquivo css em todas as páginas do seu site

Misturando os métodos

Em um mesmo documento html, é possível utilizarmos o método externo, o local e o inline. Caso isso seja feito, as linhas de código que utilizaremos no método local só valerão para a página referente àquele documento e os estilos inline apenas para a tag em que está inserido.
