Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito, mas colocar tudo no meio do par de tags p e /p

Se precisar quebrar o texto em algum lugar específico
como esse, você pode usar a tag <br>.

Ficar usando <br> para conseguir mais espaço na tela é coisa de noobão.

Para fazer comentários dentro do código, digite chave angular exclamação dois hífens. Veja o comentário secreto no código depois!

Vamos adicionar alguns símbolos especiais:
®
©
™
€
£
¥
¢
Δ
↑

Vamos adicionar alguns emoji: 🖖 🤓
Você pode até copiar e colar o emoji como a biblioteca de emojis "emojipedia" sugere. Mas é recomendado que copiemos e colemos o código U+ (ir em "technical information"), já que se apenas colado, o emoji pode não ser exibido em alguns navegadores.
Para escrever o código no VS Code, substitua "U+" por "&#x" no código. Os demais caractéres continuam do mesmo jeito.
