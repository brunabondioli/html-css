Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript.
A tag <code> pode ser usada para garantir fontes espaçadas. Como no exemplo acima!
Mesmo com códigos de programação como Python ou JS, é necessário o uso da tag <br> ou <p> para quabrar a linha/criar um novo parágrafo. A tag <code> vai apenas espaçar as letras igualmente.
A tag <pre>, entretanto, faz com que o texto obedeça as quebras de linha e espaços, de mesma maneira que o texto foi escrito no VisualCode. Incluindo tabulações, quebras de linha e espaços. Veja no exemplo:
