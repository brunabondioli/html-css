Meu Primeiro Formulário

Existem 2 métodos para envio de formulários: o "get" e o "post". No método "get", os dados coletados ficam na URL ao enviá-los. Já no método do "post", os dados não aparecem na URL, mas aparecem no DevTools.

Para proteger os dados, é necessário criptografá-los através do protocolo 'https' (estudar posteriormente).

Mas, quando usar "get" e "post"?
"GET" - dados não sensíveis (nome, peso, idade) ou quando você quer que a URL seja compartilhada. O máximo de tamanho é 3000 bytes (aprox. 3000 letras).
"POST" - dados sensíveis (melhor https), dados maiores que 3000 bytes e para enviar arquivos.
