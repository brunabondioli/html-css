Identificando/diferenciando tags

Para diferenciar (ou mais corretamente dizendo, idetificar) tags, escreva dentro das chaves angulares, após a letra ou palavra que forma a tag "id=". Em seguida, digite sua identificação; nesse exemplo "principal" (ver código tag h1). Por fim, vá até o arquivo das CSS, e após as linhas genéricas da tag em questão, digite "h1#principal" (nometag#identificaçãotag) e digite os estilos que você pretende aplicar apenas para aquela tag.
Dicas: HTML é id, CSS é #. HTML é class, CSS é . !

"Ids" Repetidos

Não é correto batizar mais do que uma tag com o mesmo nome. Ou seja, se eu já identifiquei uma tag "h1" como "id="pricipal"", eu não posso usar essa mesma idetificação em outras tags. Ao invés de "id", nesse caso, recomenda-se utilizar "class".

Escolhendo nomes de "ids" e "classes"

Sempre que for escolher uma identificação ou nome para suas tags em id ou class, pense na funcionalidade, não na forma. Veja nos parágrafos abaixo.

Usando classes no corpo do texto

Podemos modificar partes do corpo do texto com as características das classes que criamos. Veja só esse trecho com as características da classe "avaçado": "isso é apenas um exemplo!". Para isso, basta colocar o trecho do texto entre tags <span> seguida do estilo "class" e sua idetificação.

É importante ressaltar que no documento CSS, devemos registrar a class apenas com ".nomedaclasse" (Veja no documento).
