Usando links externos

Para colocar links no seu site, use a tag <a>. Dentro da tag, aparecerá o parâmetro "href=". Entre as aspas desse parâmetro, cole o endereço completo do site.
Ao clicar no link, na página pronta, o novo site se sobreporá ao site inicial. Veja:

Você pode acessar o meu repositório público no GitHub!

É recomendado que links de outros sites (externos) sejam abertos em outra guia. Para isso, dentro da tag <a>, APÓS o endereço no site, digite o parâmetro "target=" e digite "_blank" dentro das aspas e o parâmetro "rel=" e digite "external" dentro das aspas.
Basicamente esses parâmetros significam que o link é externo e que deve abrir em uma nova guia.

Usando links externos

Esta é a primeira página do site. Se você quiser, pode acessar também a minha segunda página. Para colocar o link da página 2, digite "Ctrl" + "espaço" e selecione a opção.
Importante resaltar que as páginas devem estar na mesma pasta.

Use os parâmetros "rel=next" e "rel=prev" dentro da tag <a> para indicar que a página do link da tag é a próxima ou a anterior.
Visualmente nada muda, caso o parâmetro não seja inserido. Mas, há uma questão de semântica.

Sempre que você for colocar um link de um site patrocinado, use no parâmetro "rel=" a descrição "nofollow".
Isso indicará ao Google que o site do link não deverá ser seguido, que você não dá um aval para esse conteúdo (mas o que isso significa?). Veja:

Links para download

    Para inserir um link para download, coloque o arquivo que você quer disponibilizar para download na mesma pasta da página index (pode até ser dentro de um outra pasta). Depois, procure no parâmetro "href" dentro da tag <a> o arquivo desejado, clicando em "Ctrl" + "espaço". Em seguida, digite o parâmetro "download" e complete com o nome do arquivo (ex: livropdf.pdf). Depois, digite o parâmetro "typ" ainda dentro da tag <a> e complete com o media type correto para o parâmetro (ver "iana media type" - PDF é "application/pdf" e zip é "application/zip").
